Netflix Clone

Overview

This project is a front-end clone of Netflix built to replicate the user interface and user experience of the original streaming platform. It highlights responsive web design and layout structuring skills.

Features

- Responsive homepage layout
- Banner section with featured content
- Movie thumbnails/cards section
- Navigation bar similar to Netflix
- Modern UI design

Tech Stack

- HTML5
- CSS3
- JavaScript

Project Structure

netflix-clone/
│── index.html
│── style.css
│── script.js
